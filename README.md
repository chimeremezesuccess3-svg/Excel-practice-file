# Excel Data Entry Practice

This is my first beginner-friendly data project. It demonstrates my practice in **data entry, Excel organization, and basic data management**.

## Project Description
This project contains two Excel sheets:

### 1️⃣ Student Records
- Sample data of 10 students
- Includes ID, First Name, Last Name, Age, Gender, Score, and Attendance (%)
- Used to practice **data entry accuracy**, sorting, and formatting

### 2️⃣ Sales Data
- Sample sales data of 10 products
- Includes Order ID, Product, Quantity, Unit Price, and Total
- Used to practice **basic calculations**, data organization, and Excel formulas

## Skills Practiced
- Microsoft Excel  
- Data entry and organization  
- Sorting and filtering  
- Basic formulas and calculations  

## How to Use
1. Download the Excel file: [Excel_Data_Entry_Practice.xlsx](https://github.com/success-data/excel-data-entry-practice)  
2. Open in Excel or Google Sheets  
3. Practice entering, editing, and formatting data  
4. Try adding new rows, calculating totals, or sorting columns
